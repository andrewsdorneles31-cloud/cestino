# Roadmap de Implementação

## Fase 1: Setup Inicial (Infraestrutura)
- [ ] Configuração do projeto Next.js + Tailwind.
- [ ] Criação do projeto no Supabase.
- [ ] Execução do SQL de criação de tabelas e buckets.

## Fase 2: Autenticação e Acesso
- [ ] Implementação da lógica de verificação de `codigo_unico` na Home.
- [ ] Configuração do Supabase Auth para a área `/admin`.

## Fase 3: Painel do Presenteador
- [ ] Formulários de upload de mídia (Imagens, Áudio, Vídeo).
- [ ] Integração com API de IA (Client-side).
- [ ] Lógica de persistência de dados.

## Fase 4: Experiência da Presenteada
- [ ] Construção da página emocional responsiva.
- [ ] Implementação de animações (Framer Motion).
- [ ] Verificação de status de aprovação.

## Fase 5: Administração e Finalização
- [ ] Dashboard de moderação.
- [ ] Gerador de QR Code.
- [ ] Testes de fluxo de ponta a ponta.
